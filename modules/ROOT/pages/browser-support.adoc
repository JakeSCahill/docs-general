Use Anypoint Platform with the latest version of any supported browser:

* Microsoft Edge with Chromium
* Microsoft Edge wihtout Chromium (supported through December 31, 2020)
* Google Chrome
* Mozilla Firefox
* Safari

The Anypoint Platform browser support follows the Salesforce Lightning Experience policy.

== Unsupported Browsers

Internet Explorer 11 is not supported by MuleSoft.
While the browser may work with Anypoint Platform, you may experience issues.
